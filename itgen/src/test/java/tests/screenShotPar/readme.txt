 Для запуска в режиме снятия эталонного снимка
  -- из среды разработки запускаем конфигурацию запуска
  со свойством -Detalon=true.
  -- через консоль запускать с опцией -Petalon=true.

  Пример:
  gradlew clean testScreenShots -Petalon=true   -- создатуться новые эталонные снимки для всех тестов
  gradlew clean testScreenShots                 -- тесты не создают эталонные снимки, а сравнивают с уже имеющимися.

  Для родителя добавляется -Ptarget=localPar - учетная запись под родителем
  Чтобя запустить из IDEA нудно в конфигурации запуска добавить -Dtarget=localPar
